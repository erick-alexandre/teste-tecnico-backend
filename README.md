# Desafio Backend - Nemu 

API simples para ler um arquivo Excel de jornadas de usuários e entregar os dados agrupados.


# Tecnologias

- Node.js + Express
- TypeScript
- XLSX (Para ler a planilha)


## Como rodar em 3 passos

**1. Instale as dependências:**
 - npm install

 **2. Rode o servidor:**
 - npm run dev

  **3. Consuma o endpoint:**
 - Acesse em: http://localhost:3000/journeys
